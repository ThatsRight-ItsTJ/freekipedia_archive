[edit] How to write or edit an article

Writing or editing an article (or any other page) is simple and straight forward.

  1. You'll need to sign in. If you doesn't have an account already, you can simply create one by clicking on log in / create account in the upper right corner. You'll get your account without no delay and are ready to start immediately.
  2. Click the "Edit" page tab at the top of the page you want to edit.
  3. Make changes to the text or write a new one, if there's no text yet.
    You format the page with wikimarkup, a simple language that doesn't require any programming skills. Click here for the most common formatting commands.
  4. Click the "save page" button.

That's it. For more information, strategies etc. check out the corresponding help page on MediaWiki.

[edit] How to start a new article

  1. Check, if there really isn't a corresponding article already.
  2. Consider, if you could write your article as a subtopic in an existing one.
  3. Set a link to and with the name of your new article from another, already existing page. How to add an internal link]
  4. Click on that new link to create a new article.

[edit] what is a wiki and how does it work

Basically, a wiki is website, that can be created by numerous authors in a easy and collaborative way. We use the open source MediaWiki software and information on how it works is available on there wiki.
